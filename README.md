# Instalasi package yang dibutuhkan

---
### Python 3

### nltk
pada terminal masukkan `pip install nltk` kemudian buka python di terminal dengan `python` dan jalankan program sebagai berikut:

    import nltk
    nltk.download('stopwords')
    nltk.download('punkt')

### pandas
`pip install pandas`

### flask
`pip install Flask`
`pip install WTForms`

## numpy
