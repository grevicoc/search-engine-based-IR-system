How to Use:

Search Engine
  1. Pilih tipe data dan sumber data yang ingin ditampilkan di bawah kolom pencarian (apakah dari Webscraper atau local, apakah ingin menampilkan kemunculan kata atau hasil pencarian)
  2. Masukkan query kata yang ingin anda cari
  3. Klik Search.

Upload File
  1. Klik "Upload Files"
  2. Klik "Choose Files"
  3. Pilih file yang anda ingin upload dan pastikan file ber-extension *.txt
  4. Setelah itu klik "Upload" dan file anda telah berhasil di-upload
  5. Untuk menghapus file-file sebelumnya yang telah anda upload, klik Delete Files maka semua file anda telah terhapus.
